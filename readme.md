## Execute
Execute main.py

```$ python main.py```

## Read an mjai log
You can read an mjai-log in log directory by

```File > Open```

## Play a game
You need to copy following from akochan (dev_game branch)

ai.dll (windows)  
libai.so (linux)  
system.exe  
setup_mjai.json  
params  

Then you can play a game by

```AI > Start Game```

and input random seed (non-negative integer).
